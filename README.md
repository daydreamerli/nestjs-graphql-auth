<p align="center">A Nest.js template for building with auth and mongo. Basic CRUD functionality for user. 
Lite and schema agnostic.</p>

## study project from https://www.youtube.com/watch?v=XbG1B5vjo7M&t=4s

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
